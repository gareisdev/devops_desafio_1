Alumn@: Leonel Gareis

  1. Que es un sistema de control de versiones?

    Se trata de una herramienta que permite versionar archivos, valiendose de snapshots que le permite poder volver a un momento anterior.

  2. Cual es la diferencia entre git y github?

    Git es una herramienta de versionado, mientras que GitHub es un administrador de repositorios en la nube.

  3. Cómo creo una rama en git?

    Las ramas son creadas a traves del comando: git branch [NAME] Donde [NAME] es el nombre de la rama que queremos crear.
